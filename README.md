# 🌤️ Weather App

A clean iOS Weather App built with SwiftUI.

## ✨ Features
- 🔍 Search weather by city name
- 🌡️ Display temperature in Celsius
- 💧 Humidity and wind speed info
- ⚠️ Error handling (city not found, no internet)
- ⏳ Loading state while fetching data

## 🛠 Tech Stack
- Swift
- SwiftUI
- URLSession
- Async/Await
- OpenWeatherMap API
- MVVM Architecture

## 🏗 Architecture
```
WeatherApp/
├── Models/
│   └── WeatherModel.swift
├── Services/
│   └── WeatherService.swift
├── ViewModels/
│   └── WeatherViewModel.swift
└── Views/
    └── WeatherView.swift
```

## ⚙️ Setup
1. Clone repo
2. Get API key from openweathermap.org
3. Add key to WeatherService.swift
4. Build and run!
